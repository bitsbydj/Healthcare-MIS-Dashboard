## Project Overview
This project is an interactive Healthcare MIS Dashboard created using Microsoft Excel.

## Key Features
- KPI Cards (Total Revenue, Total Patients, Paid Revenue, Pending Revenue, Avg Bill)
- Revenue by Department Analysis
- Patient Count by Department
- Doctor-wise Revenue
- City-wise Patient Distribution
- Payment Status Breakdown
- Interactive Slicers for Dynamic Filtering

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## Business Objective
To help hospital management monitor revenue performance, patient distribution, department contribution, and payment efficiency.
